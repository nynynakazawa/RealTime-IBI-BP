# 先行研究まとめと新規性の再検討（2026-06-25）

教授の指摘：「PPはこれ以上やっても新規性がない、と言い切ってよいのか。**30fps可視光・校正フリー**という前提でも本当にそう言い切れるのか。rPPG→PP研究は多く精度では勝てないだろうが、**前提が違うこと自体に価値**があるのでは。もう一度リサーチして、先行研究と自分の立ち位置をまとめ直せ。」

本MDはこの指摘に答えるための、文献の再調査結果と立ち位置の再整理。
結論を先に：**「PPに新規性なし」は言い過ぎ**だった。ただし**残る新規性は精度ではなく前提と機構の説明**にあり、しかも**直接の競合（30fpsスマホ接触PPGでPPを推定）が既に存在する**ため、新規性は「PPをカメラで取れること」ではなく、**それを取り巻く条件の束**に絞られる。以下で根拠を示す。

---

## 0. 調査の進め方

Web検索（2026-06-25実施）で、カメラ/スマホPPGによるBP・PP推定を、以下の軸で横断的に確認した。各軸で「本研究の前提（接触・30fps可視光・校正フリー・PPトラッキング・寒冷昇圧・解釈可能特徴）」と一致/相違を突き合わせる。

軸：①計測形態（遠隔=顔rPPG / 接触=指スマホ）②波長・センサ（可視RGB / NIR / 高速カメラ / 力センサ）③ターゲット（SBP/DBP絶対 / MAP / **PP**）④校正（per-subject校正 / 校正フリー）⑤評価（絶対値 / 被験者内トラッキング、random split / LOSO）⑥誘発（安静のみ / 寒冷昇圧などの摂動）⑦モデル（DLブラックボックス / 解釈可能特徴）。

---

## 1. 先行研究マップ（カテゴリ別）

### 1.1 遠隔・顔rPPG → BP（最も研究が多い領域）
- 顔動画の緑チャネル平均から脈波を取り、PTT・波形形態でSBP/DBPを推定。ROIは額・頬。([review: oaepublish chatmed 2022.025](https://www.oaepublish.com/articles/chatmed.2022.025))
- 顔の時空間マップ＋DLでの遠隔BP推定。([MDPI Sensors 23(6):2963](https://www.mdpi.com/1424-8220/23/6/2963))
- 心疾患外来患者での顔カメラBP探索。([arXiv 2503.00890](https://arxiv.org/abs/2503.00890))
- **本研究との相違**：これらは**遠隔・顔**。本研究は**接触・指**。光学的形成過程（深部血管 vs 表層微小血管）が異なり、形態特徴の意味も変わる。→ **顔rPPG研究は本研究の前提を先取りしていない。**

### 1.2 接触・スマホ指PPG → BP（本研究に最も近い領域）
- **★最重要・直接競合**：「A finger on the pulse of cardiovascular health」n=127、**30fpsスマホ接触PPG**、SBP/DBP/**PPを直接推定**、30波形指標＋SHAP解釈。([PMC11924600](https://pmc.ncbi.nlm.nih.gov/articles/PMC11924600/) / [arXiv 2401.11117](https://arxiv.org/abs/2401.11117))
  - **PP MAE 4.45 mmHg、ただし R²=0.18**（PPはSBP/DBPより相関が弱いと明記）。
  - **per-subject校正あり**（測定前後のBP実測で学習）、当初random split（過学習を自認→後に被験者層別split）、**安静のみ・摂動なし**、**絶対PP**（トラッキングではない）。結論「既存精度基準をかろうじて満たす程度」。
- スマホ指PPG波形解析の別系統でも「AAMI/BHS基準未達」「現状では医療機器の代替不可」と結論。([同上 / arXiv 2404.16049](https://arxiv.org/html/2404.16049v1))
- **本研究との相違（ここが立ち位置の核心）**：競合は **校正あり×絶対PP×安静×random/層別split×DL+SHAP**。本研究は **校正フリー×ΔPP被験者内トラッキング×寒冷昇圧×被験者単位LOSO×解釈可能2特徴の収束検証**。
  → **「30fpsスマホでPP」自体は既出。だが本研究の評価設定（校正フリー・摂動下・LOSO・トラッキング）は競合と別物**で、競合のPPは校正ありでもR²0.18と弱い。本研究のΔPP r≈0.37（p<0.01, LOSO）は同一指標での直接比較ではないが、**より厳しい設定で同等以上の追従**を示している点が主張可能。

### 1.3 寒冷昇圧 × PPG形態（本研究の機構を裏づけ／先取り注意）
- **★要注意**：寒冷刺激でのPPG形態変化を、**指接触PPG・耳たぶ・顔iPPG**で比較（n=39）。**振幅・勾配（slope）・面積が有意に低下**、PWHAは部位依存。**脈圧PPは刺激中に漸増**。指PPGで効果量が最大、iPPGは早期回復。([Frontiers Physiol 2023, 14:1127624](https://www.frontiersin.org/journals/physiology/articles/10.3389/fphys.2023.1127624/full))
  - **本研究の新特徴 sys_upslope（収縮期最大立上り勾配）→ΔPP** と方向が一致（slope変化が昇圧で動く）。**「形態勾配がPP/昇圧で動く」こと自体は既知**。
- **本研究との相違**：上記は**記述的（形態がどう変わるかの統計）**であり、専用接触センサ（高サンプリング）。**30fpsスマホで・PPトラッカーとして定量化・LOSOで汎化検証**はしていない。
  → **機構の方向性は先取りされているが、「30fps消費者機での定量的PPトラッカー化＋収束検証」は埋まっていない。**

### 1.4 オシロメトリ指圧法（＝本研究のピボット先・能動計測）
- iPhoneアプリ：指でフロントカメラ＋画面を押し、**脈波振幅の包絡**と**押圧**から**校正フリー**でBP算出。押圧は画面下ひずみゲージ（3D Touch）、**接触面積で力÷面積→指圧**を算出。SBP bias/precision −4.0/11.4、DBP −9.4/9.7 mmHg。([Nature Sci Rep 2018, s41598-018-31632-x](https://www.nature.com/articles/s41598-018-31632-x) / [Sci Transl Med 2018, aap8674](https://www.science.org/doi/10.1126/scitranslmed.aap8674))
- DBP改善（オシレーション幅変動の解析）。([PMC10640822](https://pmc.ncbi.nlm.nih.gov/articles/PMC10640822/))
- **力センサ非依存化**：専用ハード無しで指の力を推定する Vibrometric Force Estimation。([Nature Sci Rep 2024, s41598-024-75025-9](https://www.nature.com/articles/s41598-024-75025-9))
- 機械式アタッチメントでの無校正BP。([Nature Sci Rep 2023, s41598-023-34431-1](https://www.nature.com/articles/s41598-023-34431-1))
- **本研究のピボットへの含意（重要）**：**「押し込みオシロメトリで校正フリーBP」「圧の代理＝接触面積（力÷面積）」は既にMukkamala系が確立済み**。本研究が計画する「タッチ接触面積を圧代理にした押し込み」は**新規ではなく追従**。差別化するなら、(a) 力センサ/3D Touch非依存で**接触面積のみ**＋自己正規化包絡、(b) 既存の受動PP特徴と**MAP+PP→SBP/DBP再構成で合流**させるモジュール統合、(c) CNAP参照×LOSO評価、の組合せに限定される。**「押すこと自体」は売りにできない。**

### 1.5 NIR・多波長・高速カメラ（前提が違う＝本研究の制約を正当化）
- RGB-NIRカメラでのBP推定（**同じ投稿先 Artificial Life and Robotics 2025**）。([Springer s10015-025-01015-3](https://link.springer.com/article/10.1007/s10015-025-01015-3)) → **NIR多波長**前提。
- 30fpsはPTTの時間分解能（±16.6ms）に不利、≥100fps高速カメラが有利と複数文献。([researchgate frame rate 2015] ほか, 検索結果群)
- **本研究との相違**：本研究は**可視RGB単一・30fps・PTT非依存（単一部位形態）**。NIR/高速/多波長を**あえて使わない**制約こそが前提の差。→ **これらは本研究を否定せず、むしろ「最も入手容易な構成での到達点」という枠を正当化**する。

### 1.6 評価方法論・校正フリーの落とし穴（本研究の誠実さが効く点）
- PPG-BPの評価基準・報告指針。([Nature Comms Med 2024, s43856-024-00555-2](https://www.nature.com/articles/s43856-024-00555-2))
- 校正フリーの楽観的結果の多くは**MIMIC等でのデータリーク由来**で過大評価、と批判。([arXiv 2404.16049](https://arxiv.org/html/2404.16049v1))
- **本研究との関係**：本研究は**被験者単位LOSO＋CNAP参照＋アーティファクト限定除去（生理的大変動を消さない）**で、この批判をむしろ回避する側にいる。**「正直に校正フリーをやると弱い」を機構で説明**している点は、リーク常習の文献群に対する価値になりうる。

## 3. 自分の立ち位置の再整理（正直版）

### 3.1 教授の指摘は妥当 ―「新規性なし」は言い過ぎだった
- これまでの「PPは既出だから新規性なし」は、**顔rPPG中心の判断**で、**接触・30fps・校正フリー・摂動下トラッキング**という前提差を潰しすぎていた。
- 軸ごとに見ると、**◯が4つ（LOSO校正フリー／PP-MAP機構説明／摂動下ΔPPトラッキング／解釈可能特徴の収束検証）**残る。**ここに価値がある**というのが文献調査の結論。

### 3.2 ただし冷静に ― 売りにできないものを誤認しない
- **「30fpsスマホ接触PPGでPPを取れる」自体は既出**（PMC11924600）。ここを新規性の核に置くと即否定される。
- **「押し込みオシロメトリ＋接触面積」も既出**（Mukkamala系）。ピボット先の「押すこと」「面積=圧代理」も新規ではない。
- **精度では勝てない**：DL/NIR/高速カメラ/校正ありのSOTAに、n=12・可視30fps・校正フリーで勝てる見込みはない。**精度noveltyは狙わない。**

### 3.3 では何を主張するか ―「精度」ではなく「前提×機構×誠実さ」
本研究の貢献を、accuracy contribution ではなく **feasibility / mechanistic contribution** として明確化する：

1. **前提の極限化**：NIR・高速・多波長・力センサ・per-subject校正を**すべて外した最小構成（可視30fps・接触・無校正）**で、何が物理的に回復可能かを切り分けた。→ 回復可能なのは**PP（脈圧）であってMAP（平均圧）・絶対BPではない**ことを、PP/MAP分解で機構的に示した。これは**negative result（絶対BP不能）を「なぜ」で説明する**positive貢献。
2. **より厳しい評価設定での追従提示**：競合（校正あり・安静・絶対PP）に対し、**校正フリー・寒冷昇圧・被験者内ΔPP・LOSO**という、リークを排し汎化を問う設定で ΔPP r≈0.37（p<0.01）を示した。指標は直接同一でないが、**「厳しい設定での到達点」**として位置づく。
3. **解釈可能特徴の収束検証**：cosφ（非対称Sinモデル）と sys_upslope（生波形微分）が**独立に同一の源（収縮期立ち上がり形）**を指し（両者Δ相関0.60）、ブラックボックスDLに依らず機構を同定した。
4. **評価の誠実さ**：アーティファクト限定除去（**生理的大変動＝信号を消さない**）で、統計的外れ値除去が手法比較を反転させる落とし穴を回避。校正フリーのリーク批判文献に対し、**正直にやると弱いが、その弱さの機構を説明できる**側に立つ。

### 3.4 残るリスク（教授に正直に言うべきこと）
- 上記4点は**いずれも「中規模のfeasibility/機構貢献」**であり、単独で強い新規性とは言いにくい。**束ねて初めて一本の物語**になる。
- **n=12は被験者間・絶対値の統計に非力**。ΔPP r≈0.37も決定的ではない。
- 真に跳ねるには、ピボット（押し込みでMAP軸を能動取得→受動PPと合流して絶対SBP/DBP）の**新データでの成立が必要**。ただし押し込み自体は既出なので、**差別化は「無力センサ・接触面積のみ・受動PPとのモジュール合流・CNAP×LOSO」に限定**して主張する。

## 4. 次アクション（リサーチ面）

1. **直接競合の精読**：PMC11924600 を全文精読し、(a) PPの被験者層別split時のR²、(b) 特徴セット、(c) 本研究のΔPPトラッキングと**指標を揃えて比較可能か**を確定。論文では「同一指標でない」を明記して不当比較を避ける。
2. **寒冷昇圧×形態の確定**：Frontiers2023 を全文精読し、slope/amplitude/area↔PP の効果量を引用可能な形で押さえ、**本研究のsys_upslopeを「30fpsスマホでの定量トラッカー化」として差分提示**。
3. **押し込み系の確定**：Mukkamala系（Sci Transl Med 2018 / Sci Rep 2018,2023,2024）を読み、**接触面積＝力÷面積の扱い**を確認。本研究の「接触面積のみ・自己正規化包絡・無力センサ」が**どこまで差分か**を線引き。
4. **投稿先内の競合**：AROB 2025 RGB-NIR論文（[s10015-025-01015-3](https://link.springer.com/article/10.1007/s10015-025-01015-3)）を確認し、**同誌でのBP研究の水準と差別化軸（可視のみ vs NIR）**を把握。
5. **評価基準の順守**：Comms Med 2024 の報告指針に沿って、本研究の主張（PPトラッキング・校正フリー・LOSO）を基準準拠の形で記述。

## 5. 一行サマリ
**「PPに新規性なし」は撤回。残る新規性は精度ではなく〈可視30fps・接触・校正フリー・寒冷昇圧下ΔPPトラッキング・LOSO〉という前提の束と、PP/MAP分解による機構的説明・解釈可能特徴の収束検証にある。ただし〈スマホ接触PPGでPP〉も〈押し込みオシロメトリ＋接触面積〉も既出のため、主張は「条件の束＋機構＋誠実な評価」に厳密に限定し、精度競争には踏み込まない。**
