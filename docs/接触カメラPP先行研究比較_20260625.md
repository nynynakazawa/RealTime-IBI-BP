# 触型（可視）カメラ → 脈圧PP：先行研究の精査と自手法の評価（2026-06-25）

対象を**接触型カメラ（指をカメラに当てる／可視光、スマホに限らず）で脈圧PP（または相対PP変化）を扱う研究**に厳密に絞って再調査した。各研究の**カメラ仕様・fps・解像度・参照・校正・PP結果**を表で揃え、最後に**自手法の精度を同じ土俵で評価**する。

結論（先出し）：

- **接触可視カメラでPPを「絶対値推定」した直接競合は実質1本**（Xiaomi 30fps, n=127）。ただし**校正フリーにすると PP R²=0.18 と弱い**（＝接触カメラPPは皆が苦戦）。
- **接触ビデオ×寒冷昇圧×相対PP**をやった研究はあるが、**レーザースペックル＋2波長＋80fps CMOS、n≈10、推定モデルなし（振幅の記述比較のみ）**で、本研究とは前提が違う。
- **〈可視・単波長・30fps・スマホ接触・校正フリーLOSO・寒冷昇圧・被験者内ΔPPトラッカー・解釈可能特徴〉という束を満たす研究は見当たらない**。＝前提の差に新規性は残る。ただし**精度では競合に勝てず、自手法のΔPP r≈0.37も「弱いが有意」レベル**。

## 2. 各研究の要点（接触可視カメラ×PP視点）

### ① A finger on the pulse（最も近い直接競合）

- **接触・可視・30fps・スマホ**という前提が本研究とほぼ一致し、かつ**PPを明示ターゲット**にした唯一の絶対値推定。([BioMedEng OnLine 2025](https://link.springer.com/article/10.1186/s12938-025-01365-w) / [PMC11924600](https://pmc.ncbi.nlm.nih.gov/articles/PMC11924600/))
- 仕様：Xiaomi Mi-8 SE 背面＋フラッシュ、30fps、120×160px、4分×6本、n=127、有効657サンプル、Omron参照。
- 校正フリー（測定前後の平均BPで学習）、被験者層別80:20、**安静のみ**。
- **PP結果：MAE 4.45–4.49 mmHg だが R²=0.18**（SBP/DBPより明確に弱いと記載）。bias 0.46±5.44。
- **示唆**：MAEが小さく見えるのは PP のレンジが狭い（健常安静）ため。**R²=0.18 は「校正フリー・安静でも PP の追従はほとんど効いていない」ことを意味**する。＝接触可視カメラでPPは皆が苦戦。

### ② SPG vs PPG under CPT（接触指ビデオ×寒冷昇圧の最近接）

- **指接触・ビデオ・寒冷昇圧・相対PP**という軸が本研究と重なる唯一の研究。([Sensors 2023, 23(11):5016](https://www.mdpi.com/1424-8220/23/11/5016) / [PMC10255579](https://pmc.ncbi.nlm.nih.gov/articles/PMC10255579/))
- 仕様：CMOS 80fps（多重化で実効40fps/波長）、640×400、639nm+850nmレーザ各10mW、指クリップ~1cm、n≈10、Finapres参照。
- **ただし**：①レーザースペックル(SPG)＋PPGの**2波長・専用光学系**（消費者スマホではない）、②**PP推定器を作らず**、CPTでの**AC振幅・SNRの低下を記述比較**するのみ。参照PPは CPTで +9%±10%。
- **示唆**：本研究の「30fps可視・単波長スマホで、ΔPPを特徴量で定量追従」は**ここでも埋まっていない**。彼らは高Fs・多波長・専用光学でも「PP値の推定」には踏み込んでいない。

### ③ Cold stress × PPG形態（機構の裏づけ／一部先取り）

- 寒冷昇圧で **slope・振幅・面積が有意低下、PPは漸増**。([Front Physiol 2023, 14:1127624](https://www.frontiersin.org/journals/physiology/articles/10.3389/fphys.2023.1127624/full))
- 本研究の **sys_upslope（収縮期勾配）→ΔPP** と方向一致。**「勾配がPP/昇圧で動く」のは既知**。
- ただし接触側は**専用PPGセンサ（カメラではない）**、かつ**記述統計でPP推定器ではない**。本研究の差分＝「30fps消費者カメラでトラッカー化・LOSO汎化」。

### ④ Oscillometric finger pressing（能動・ピボット先）

- 指で前面カメラ＋画面を押し、包絡＋押圧から校正フリーBP。([Sci Transl Med 2018](https://www.science.org/doi/10.1126/scitranslmed.aap8674) / [Sci Rep 2018](https://www.nature.com/articles/s41598-018-31632-x))
- **PPを分離せずSBP/DBP**。能動押込みなので受動形態とは別系統。本研究のピボット先だが「押込み・接触面積=力÷面積」は**既出**。

## 3. 自手法の精度評価（同じ土俵で正直に）

### 3.1 数値の対応関係（指標が違う点に注意）

| 観点         | 競合① (接触可視30fps)                                                   | 本研究                                                             |
| ------------ | ------------------------------------------------------------------------ | ------------------------------------------------------------------ |
| PP指標       | **絶対PPのR²=0.18**（横断、校正フリー、安静、層別split）          | **被験者内ΔPPのr=0.37**（縦断トラッキング、LOSO、寒冷昇圧） |
| 直接比較可否 | △**指標が非同一**（横断R² vs 被験者内Δr）。単純な優劣比較は不当 |                                                                    |

- 競合①の R²=0.18 は **r≈0.42 相当**だが、これは被験者間ばらつきを含む**横断**相関で、しかも**安静（PP変動が小さい）**条件。
- 本研究の r=0.37 は **被験者内のΔPP（昇圧で誘発した変動）への追従**で、**校正フリーLOSO**かつ**摂動下**。**より厳しい設定**での弱め～中程度の追従。
- → **「勝った」とは言えない**が、**「競合が測っていない設定（校正フリー×摂動下×被験者内トラッキング）で、同程度の弱い追従を示した」**とは言える。

### 3.2 自手法の精度の率直な評価

- **絶対値（SBP/DBP）**：MAE ~15/12 mmHg、相関≈0、MAP r≈0。**絶対値は完全に不成立**（PP/MAP分解で機構説明済み）。AAMI/BHS基準は全く満たさない。
- **PPトラッキング**：r=0.37（p=0.002）、CCC=0.29、符号一致82–91%。**有意だが弱い**。CCC<0.3 は一致度として低い。臨床利用水準ではない。
- **統計的検出力**：n=12・2,467拍。被験者数が少なく、被験者間・絶対値の主張は非力。r=0.37も信頼区間は広いはず。
- **競合との精度差**：DL/多特徴/校正あり（競合①は34–136特徴＋RF）に対し、本研究は**単一の解釈可能特徴**。精度の絶対水準では競合に及ばない見込み。

### 3.3 では本研究の精度は「無価値」か → No、ただし条件付き

- **精度競争では負ける**。これは率直に認める。
- しかし**「弱い」こと自体が機構的発見の一部**：競合①も校正フリーでは PP R²=0.18 と弱く、**接触可視カメラでPPが弱いのは本研究固有の失敗ではなく領域共通の壁**。本研究はその壁を **PP/MAP分解で「なぜ弱いか（MAP軸が無いから絶対は不能、回復可能なのはPP軸のみ）」と機構説明**している。
- **設定の厳しさが価値**：競合が避けた**校正フリー×LOSO×摂動下×被験者内トラッキング**でも**PP軸だけは符号一致82–91%で追える**ことを、**解釈可能な単一特徴**で示した。＝精度ではなく**「最小構成で何が物理的に回復可能か」の境界を引いた**。

## 4. 総合評価：教授の問いへの回答

> 「30fps可視・校正フリーでもPPに新規性なしと言い切れるのか」

- **言い切れない**。接触可視カメラで**PPを扱った研究は①③のみ**で、①は**校正フリーにすると弱い（R²0.18）・安静のみ・絶対値**、③は**専用センサの記述統計**。
- **本研究の前提の束**＝〈可視・単波長・30fps・スマホ接触・**校正フリーLOSO**・**寒冷昇圧**・**被験者内ΔPPトラッキング**・**解釈可能特徴の収束検証**〉は、**どの先行研究も同時に満たしていない**。→ **前提差に新規性は残る**。
- **ただし新規性の質は「精度」ではない**：
  - 精度では①（校正あり・多特徴RF）にもSOTA（NIR/高速/DL）にも勝てない。**accuracy noveltyは放棄**。
  - 主張すべきは **method/feasibility novelty**：①「最小構成で回復可能なのはPPでありMAP/絶対ではない」を**PP/MAP分解で機構提示**、②校正フリー×摂動下×LOSOという**厳しい設定での追従可能性**、③**解釈可能特徴（cosφ≡sys_upslope）の収束検証**、④**生理的大変動を消さないアーティファクト限定除去**という評価の誠実さ。

### 4.1 論文化での具体的な書き方（推奨）

- **比較表で①を正面に置く**：「接触可視30fpsでPPを推定した先行研究①は、校正ありでもPP R²=0.18。本研究は校正フリー・摂動下・被験者内で r=0.37 を**異なる指標で**示す」と**指標の非同一性を明記**して不当比較を避ける。
- **②③を「機構の傍証」として引く**：寒冷昇圧でslope/PPが動くのは既知（③）、専用光学でもPP推定器は未構築（②）→「本研究は消費者30fps単波長で**PPトラッカーを構築**した点が増分」。
- **限界を先に書く**：n=12、r=0.37は弱い、絶対値不能、臨床水準でない、を**自分から明示**。誠実さが①の校正フリー弱さと対比されて効く。

## 4.5 補足A：MAE/MAPEを含めて評価するとどうなるか（2026-06-26追記）

問い：「相関だけでなく MAE/MAPE の精度も含めて評価したら、自手法はどう見えるか」。
結論：**MAE/MAPEを足しても自手法の優位は出ない。むしろ競合①の土俵に乗って不利になる。** PPでMAE/MAPEを主指標にしてはいけない。

### 4.5.1 PP特有の罠：MAEは「平均予測」でほぼ達成される

- PPはレンジが狭い（健常で概ね30–50 mmHg、変動はさらに小さい）。→ **MAE/MAPEは構造的に小さく出るが、中身は「被験者平均を出すだけ」のベースラインでほぼ説明できる**。
- 競合①がこれを自分で証明している：PP **MAE 4.45 / R²=0.18 / bias 0.46±5.44**。
  - R²=0.18 → 残差SD = SD_PP×√(1−0.18) = SD_PP×**0.91**。**「被験者平均予測」から残差を9%しか縮めていない**。MAE 4.45 のほぼ全部はベースライン由来。
  - PP≈45なら **MAPE≈10%** だが、これも中身はベースライン。
- 教訓：**PPのMAE/MAPEは、必ず「被験者平均予測ベースラインのMAE/MAPE」と並記し、その改善幅（ΔMAE, % reduction）で語らないと無意味**。単独数字は誤誘導。

### 4.5.2 自手法をMAE/MAPEで出した場合の見積もり

- 自手法の **PP絶対MAEは現状未計算（要算出）**。ただしΔPP r=0.37 → R²=0.14。被験者内ΔPP分散の14%しか説明していない → 残差SD = SD_ΔPP×√(1−0.14) = ×**0.93**。**ベースライン改善は競合①と同程度に小さい**見込み。
- 寒冷昇圧でPPは +9%±10%（研究②）程度しか動かない → ΔPP MAEも数mmHg・MAPE一桁〜十数%に収束し、**競合①と同じ「低分散で小さく見える」帯**に入る。

### 4.5.3 含意（評価設計の方針）

- **MAEで正面比較すると競合①が有利**（絶対PP MAE 4.45 という単一の見栄えする数字を持つ）。自手法はMAE土俵では差を出せない。
- → **見出しはトラッキング指標（r / CCC / 符号一致）のまま維持**。MAE/MAPEは「**被験者平均ベースラインとの改善幅**」とセットでのみ提示し、**誠実さの担保**として扱う（主指標に格上げしない）。
- 宿題：PP絶対MAE・ΔPP MAE/MAPE を**被験者平均ベースライン併記**でLOSO算出する（論文化前に必須。Codexへ委譲可）。

## 4.6 補足B：接触面積MAP（押し込みオシロメトリ）と統合したら新規性は上がるか（2026-06-26追記）

問い：「受動PP＋接触面積由来MAP（押し込み）で結果が出れば、より新規性になるか」。
結論：**大きく上がる。ただしフレーミング次第で、間違えると即潰される。**

### 4.6.1 弱いフレーミング（不可）
「押し込みでMAP、受動形態でPP、足してSBP/DBP」
→ 査読者「オシロメトリ④（Mukkamala）は**押すだけでSBP/DBPを丸ごと**校正フリーで出す。あなたのPPは冗長では？」で終了。④は既に押込のみで絶対BPを出している。**“押すこと”“接触面積=力÷面積”は既出**（[本表④](#1-比較表接触可視カメラ系--pp)）。

### 4.6.2 強いフレーミング（これなら新規）

**「押し＝MAP/絶対軸の“自己校正アンカー（たまに）”、受動形態＝その間の連続PP/BP変化トラッキング」というハイブリッド時間アーキテクチャ。**
- 時々押して絶対値（MAP軸）をアンカーし、**押していない間は受動PPでSBP/DBPの変化を連続追従**。
- 外部カフ不要・**力センサ不要（接触面積のみ）**・消費者30fps可視・単波長。
- **PP/MAP分解の機構**が「なぜこの役割分担か（受動はPP軸しか持たない＝MAPは押しで補う必然）」を理論的に正当化。④もNIRもこの**役割分担の機構説明＋連続トラッキング**は持っていない。
- 位置づけ：**force-sensorレス・カフレスの “continuous cuffless monitoring with periodic self-calibration”** という未占有ニッチ。

### 4.6.3 新規性成立の必要条件（査読で必ず突かれる3点）

1. **押し包絡のピーク ↔ CNAPのMAP が対応**する（相対オシロメトリの成立）。
2. **押しMAP＋受動PP→絶対SBP/DBP が AAMI/BHS に近づく**（少なくとも受動単独より明確改善）。
3. **受動PPが「押し単独」に対し連続区間で付加価値を持つ**ことを示す（押しの間を埋める価値）。**これが無いとPPは冗長**で②の意味が消える＝最重要。

### 4.6.4 判定

- 3点（特に3）を実データで示せれば、**「接触可視カメラでPPだけ」よりはるかに強い貢献**になる（前提の束＋機構＋連続自己校正アーキテクチャ）。
- ただし**新規性の核は“押すこと”ではなく**、(a) 力センサレス・接触面積のみ、(b) 受動PPと押しMAPのモジュール合流（MAP+⅔PP）、(c) PP/MAP分解による役割分担の機構的正当化、(d) 連続追従と周期的自己校正の時間設計、の**組合せ**に厳密限定して主張する。

## 5. 一行サマリ

**接触可視カメラでPPを推定した直接競合は①（30fpsスマホ）のみで、校正フリーではPP R²=0.18と弱い。寒冷昇圧×指ビデオの②は2波長レーザ80fpsでPP推定器なし。本研究の〈30fps単波長スマホ・校正フリーLOSO・寒冷昇圧・被験者内ΔPPトラッカー・解釈可能特徴〉の束はどこも満たしておらず前提差に新規性は残るが、自手法のΔPP r=0.37は「弱いが有意」で精度競争には勝てない。主張は精度ではなく〈最小構成での回復可能境界の機構的提示＋厳しい設定での追従＋誠実な評価〉に置く。**
