30fps可視光スマホPPGによる血圧推定研究の、絶対値BPの限界の特定 → PP追従の確立 → 外れ値方法論 → 新規性判定 → 押し込みオシロメトリへの転換、までの記録。

## 1. 出発点：6手法の絶対値BP（既存）

- 3手法 × {Direct回帰, Baseline+Tracking回帰} = 6系列。
  - RTBP(M1): A, HR, V2P_relTTP, P2V_relTTP（純・脈波形態）
  - SinBP_D(M2): RTBP特徴 + 残差E（非対称SinフィットのRMS残差＝波形歪み）
  - SinBP_M(M3): A, HR, Mean, sinφ, cosφ（非対称Sin波パラメータ）
- 全特徴は緑チャネルrPPG由来の拍ごと形態量。CNAP(SYS/DIA-direct)を参照、被験者単位LOSO、20秒窓。
- 評価PG: `Analysis/bsn_absolute_bp/run_current_loso_base_variants.py`（本セッションで128→6系列にリファクタ、ridge a=1.0, gamma=1.0）。

## 2. 主要な発見（時系列）

### 2.1 絶対値は追えない／PPは追える（PP/MAP分解）

- 被験者間の絶対レベル（ベースライン）は校正なしでは復元不能（between MAE ~10–15 mmHg）。
- 被験者内トラッキング：**PP（脈圧）は追える、MAP（平均圧）は追えない**。
- 機構：緑PPGは脈波の「形」＝脈圧軸しか持たず、平均圧軸（オフセット/光学DCで未較正）を持たない。
- SBP=MAP+⅔PP, DBP=MAP−⅓PP はMAP軸が主成分 → SBP/DBPは追えない。

### 2.2 ridgeでは目標表現が等価

- Direct回帰では「MAP+PP→合成」と「SBP/DBP直接回帰」は**数学的に同一**（ridgeの線形性）。
- → 目標表現は梃子にならない。梃子は特徴量側。

### 2.3 特徴量スクリーニング（被験者内Δ相関）

- 有意トラッカー：HR→DBP(r=0.29,p=0.03)/MAP(r=0.27,p=0.016)、**cosφ→PP(r=0.36,p=0.01)**。
- SBPは全特徴で≈0。M2_Stiffness_used=0固定（死に列）、M2_A_usedは被験者内ほぼ定数。

### 2.4 新・収縮期特徴（生波形から抽出）

- スマホから13セッションの緑生波形(Wave_Data: 経過時間_秒,Green,SinWave, ~30fps)を抽出。
- 新特徴 **sys_upslope_max（収縮期最大立上り勾配, max dGreen/dt）** を定義。
- **sys_upslope_max → ΔPP: r=0.37, CCC=0.29, p=0.002, 符号一致82–91%**（これまでで最良のPPトラッカー、30fps固有）。
- ただしSBP/DBP/MAPは新特徴でも≈0。

### 2.5 組み合わせはSBPを救わない

- HR(MAP側)＋sys_upslope(PP側)の被験者内Δ多変量でもSBPは≈0/負。n=12で多変量Δは汎化せず。
- cosφ＋upslopeは冗長（Δ相関0.60）で結合は単体に劣る → **sys_upslope単体**が最良。

### 2.6 外れ値方法論（重要な反転）

- 素朴なMAD外れ値除去は「rejection=on」で従来法RTBPが勝つ結果を出したが…
- 診断：**mad_bothが消した窓は高ΔBP窓（mean|z_refPP|=2.8 vs 残0.7）＝寒冷昇圧の生理的大変動＝信号そのもの**。
- artifact_only（特徴は極端だが参照は平常な窓のみ除去、生理的大変動は残す）では**提案(upslope/cosφ)が従来法を上回る**（PP r=0.37, p<0.01）。
- 教訓：BPトラッキング評価では統計的外れ値除去が信号を消し手法比較を反転させる。コヒーレンスに基づくアーティファクト限定除去が必要。

### 2.7 fpsスケーリング（前者/後者の決着）

- 30fps生波形を間引き(30→15→10→7.5fps)。factor=1で専用PGと完全一致(0.391)を確認後に判定。
- **damageは低fpsで増えず（30fpsで最大）、quant_corr≈0**。Nyquist上7.5/10fpsは波形崩壊で無効、有効点は実質30/15の2点のみ。
- 結論：「30fps特有の前処理」という強い新規性（前者）は**立証できず**。外れ値知見は一般原則（後者）寄り。

## 3. 文献調査と新規性判定

- カメラ/rPPGでの脈波/脈圧/動脈硬化推定は**既存**（BSPC2020 高速カメラ, palmar 10波長2022 等）。
- 較正フリー・スマホBPも既存（Nature Sci.Rep.2018 で R>0.70）。寒冷昇圧BP変化トラッキングも既存。
- 判定：「カメラでPP」単独は新規性弱。精度novelty困難（n=12, r=0.37はSOTA未満）。
- 残る筋：制約ニッチ（30fps較正フリー解釈可能特徴）＋外れ値方法論＋PP/MAP分解の束 = 中規模feasibility貢献。強い新規性には追加データ/別軸が必要。

## 4. 方向転換：押し込みオシロメトリ＋多チャネル（能動計測）

- 緑PPG（受動）はPPしか取れないと確定 → 「形を見る」から「**圧をかけて測る**」へ。
- **押し込みオシロメトリ**：指でフロントカメラ＋画面緑照射を徐々に強く押す。脈波振幅の包絡がMAPでピーク（カフのオシロメトリ同原理）。圧源＝自分の指＝カフ不要。**欠けたMAP/絶対値軸に直撃**。
- 圧の代理：**タッチ接触面積**（MotionEvent標準API, 全機種）。力センサ不要。
- 絶対mmHg化の方針：機種ごと力較正は不採用（可搬性が壊れる）。**ガイド一定速度押し込みで再現性を上げ、CNAP参照で学習**。
- 個人差対策：接触面積は計測内で自己正規化（包絡の形は指サイズ非依存）、光学アンカー（振幅ピーク=MAP, 脈波消失=SBP）、DC遮光を面積より血管寄りの圧代理に。
- 追加軸：RGB（多波長, 副軸）、接触面積。
- UI制約：カメラ穴はステータスバー帯 → 穴の上に描かず、穴の少し下にランディングゾーン＋緑リング、覆い判定はPPG拍動、位置はタッチ重心オフセットで。

## 6. 結論と未決事項

- **確定**：受動緑PPGはPPのみ追従可（sys_upslope/cosφ, p<0.01）、MAP/絶対は不可。SBP/DBP校正フリー追従は構造的に不能。
- **未確定（要新データ）**：押し込みオシロメトリでMAP/絶対が取れるか。まず相対（包絡ピーク↔CNAP MAP）の成立をde-risk。
- **新規性**：現状は中規模。押し込みでMAP軸を取れれば「受動PP＋能動MAP→SBP/DBP」で跳ねる可能性。

## 8. 過去研究（RTBP/SinBP_M/SinBP_D）の位置づけ

過去研究は「否定」ではなく「踏み台」として位置づけ直す。

### 8.1 各手法の再定義

| 過去の手法                | 新しい位置づけ                                                                                        |
| ------------------------- | ----------------------------------------------------------------------------------------------------- |
| RTBP                      | 従来形態ベースライン（比較基準）                                                                      |
| **SinBP_M (cosφ)** | PP追従源（収縮期立ち上がり形）を**モデルベースで発見**。新 sys_upslope と**収束的に検証** |
| **SinBP_D (E)**     | 残差アイデア。正直に「**独自寄与は弱かった**」と報告する比較対象（誠実さの担保）                |
| 共通の発見                | **PPは追える/MAPは追えない**＝発想転換の論拠                                                    |

### 8.2 cosφ と sys_upslope は「収束的検証」（無駄ではない）

- cosφ→ΔPP r=0.36(p=0.01)、sys_upslope→ΔPP r=0.37(p=0.002)。**ほぼ互角**。
- 両者の被験者内Δ相関＝0.60（冗長）＝**同一現象（収縮期立ち上がり形→PP）を別の数学で記述**。
- 異なる2手法が独立に同じ源を同定 → 主張がむしろ強くなる（convergent validation）。
- SinBP_M は「コンパクト全波形モデルでこの源を発見した側」として温存。

### 8.3 SinBP_D の過去主張は「嘘」ではない（指標の区別）

- 過去論文の主張は **centered MAE/RMSE（被験者内トラッキング誤差の小ささ）** の「favorable tendency」。
- 「absolute は不十分」「相関は大SD・補助的・not conclusive」と**明示的にヘッジ済み**＝絶対値・強い追従は主張していない。
- centered MAE（誤差の大きさ）と 被験者内Δ相関（追従の方向・一致）は**直交**。E単独の相関が弱い(r=0.145,p=0.35)ことと矛盾しない。
- → **撤回不要**。今後の注意：SinBP_Dの小さな centered MAE 差を「追従改善」と言い換えない。外れ値知見からも当時の小差は慎重に読む（論文自身の n小・大SD ヘッジと整合）。

### 8.4 論文の物語の弧（推奨構成）

1. 従来研究：表現を比較、トラッキングは弱く補助的・絶対は不可、と**正直に報告（ただし"なぜ"は未解明）**。
2. 本研究：PP/MAP分解で「カメラは脈圧は持つが平均圧を持たない」と示し、**弱い相関の理由を機構的に解明**。追従源を収縮期立ち上がりに局在化（cosφ＝sys_upslope）。
3. 発想転換：だから**PP追従に着目**＋**能動押し込みでMAP軸を取りに行く**。
   → negative result を "なぜ" で説明して前進する構成。過去を否定せず踏み台にする。
