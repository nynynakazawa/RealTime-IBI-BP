## スライド2：絶対値SBP/DBPは合わなかった

**メッセージ：6系列とも絶対誤差が大きく相関もほぼ0。さらにB+Tの悪化は「被験者間レベル」に集中していた**

| 手法         | 回帰   | SBP：MAE±SD ／ r±SD      | DBP：MAE±SD ／ r±SD      |
| ------------ | ------ | -------------------------- | -------------------------- |
| RTBP（従来） | Direct | 15.3±10.9 ／ −0.09±0.48 | 12.1±8.6 ／ +0.14±0.49   |
| SinBP_M      | Direct | 15.1±11.4 ／ −0.11±0.52 | 11.6±9.2 ／ +0.01±0.54   |
| SinBP_D      | Direct | 15.0±11.1 ／ −0.17±0.62 | 12.3±8.9 ／ −0.06±0.67  |
| RTBP（従来） | B+T    | 20.6±11.7 ／ −0.02±0.38 | 19.7±15.3 ／ +0.17±0.51  |
| SinBP_M      | B+T    | 19.7±14.3 ／ −0.28±0.55 | 22.3±19.3 ／ +0.06±0.52  |
| SinBP_D      | B+T    | 16.2±7.8 ／ −0.24±0.44  | 12.8±10.7 ／ −0.00±0.54 |

- Directで SBP MAE 約15・DBP 約12、相関は負〜0
- **B+Tはむしろ悪化**。誤差を分けると、悪化は **被験者間レベル（between）に集中**：MAP betweenが **12→20** に増える一方、被験者内変動（within）は **3.3 のまま不変**
- つまり外しているのは「その人の平均的な圧＝レベル」＝**MAP成分が合っていない**疑い
- 相関のSDが大きい（≈0.5）のは被験者間ばらつき大（n小の限界）

→ ならば、MAPとPPに分けて確かめてみる

## スライド3：分けて見ると、MAPは追従せず・PPだけ追従

**メッセージ：仮説どおり、平均圧MAPは追従していない。脈圧PPだけが追従していた**

| 手法         | 回帰   | MAP：r±SD   | PP：r±SD             |
| ------------ | ------ | ------------ | --------------------- |
| RTBP（従来） | Direct | +0.01±0.51  | +0.21±0.40           |
| SinBP_M      | Direct | −0.03±0.58 | **+0.39±0.46** |
| SinBP_D      | Direct | −0.11±0.64 | +0.07±0.62           |
| RTBP（従来） | B+T    | +0.01±0.57  | +0.27±0.44           |
| SinBP_M      | B+T    | −0.07±0.53 | +0.14±0.53           |
| SinBP_D      | B+T    | −0.08±0.51 | +0.23±0.48           |

![図1：被験者内トラッキング相関 by target（誤差棒=被験者間SD。MAP≈0／PP高）](../../Analysis/Data/bsn_absolute_bp/slide_figures/fig1_tracking_by_target.png)

（補助図：散布図 `fig2_scatter_map_vs_pp.png`、被験者jinの時系列 `fig3_timeline_subject.png`）

- **MAPの相関は6系列すべて ≈0**（−0.11〜+0.01）＝レベルを全く追えていない
- **PPは正の相関**（Direct/SinBP_M で +0.39）＝脈圧は追えている
- SBP=MAP+⅔PP だから、**MAPが取れない時点でSBP/DBPの絶対値は取れない**。B+Tの悪化もこのMAP不全の表れ

→ では、その「追えているPP」は、どの特徴が担っているのか

## スライド4：PPを追えているのは「収縮期の立ち上がり」

**メッセージ：既存のcosφと、新しい収縮期勾配が、独立に同じ源（収縮期立ち上がり）を指した**

| 特徴                                     | → ΔPP 相関 r±SD | p値   |
| ---------------------------------------- | ------------------ | ----- |
| cosφ（既存SinBP_M）                     | 0.42±0.31         | 0.002 |
| sys_upslope（新規・収縮期勾配/微分使用） | 0.37±0.30         | 0.002 |
| P2V_relTTP（既存RTBP、下降時間）         | 0.25±0.31         | 0.04  |

![図4：PP追従に効く特徴（誤差棒=被験者間SD。cosφ・収縮期勾配が最上位, p<0.01）](../../Analysis/Data/bsn_absolute_bp/slide_figures/fig4_pp_feature_screening.png)

- 被験者内Δ相関で **PPは有意**（p<0.01）、一方SBPは最良の特徴でも有意でない
- cosφ と sys_upslope は実質同じ現象を別の数式で見ている（両者の相関0.60）＝**異なる2手法が独立に同じ結論**
- 過去研究で「強い相関が出なかった理由」も、ここで腑に落ちる（追えるのはPPで、SBP/DBPはMAP不全に引っ張られていた）

→ ここまでで、「受動的に波形の形を見る」アプローチの限界が見えた

## スライド5：「形を見る」から「圧をかけて測る」へ

**メッセージ：受動ではPPしか取れない。だから押し込みで、欠けているMAP軸を能動的に取りに行く**

- 受動の限界：カメラは脈波の形（PP）は持つが、平均圧（MAP）の情報を持たない
- 文献的にも、カメラ/rPPGでの脈波・脈圧推定は既出 → 受動だけでは頭打ち
- 発想転換：指でカメラ＋画面緑照射を徐々に押すと、**脈波振幅の包絡がMAPでピーク**になる（カフのオシロメトリと同原理・カフ不要）
- 圧の代理は **タッチ接触面積**（標準API・ほぼ全機種）。得たMAPは既存の MAP+PP→SBP/DBP 式に差し込めるので、**これまでの研究は捨てずPP成分として活かす**

→ 今週は、この新しい計測のための土台を実装した

- これが出れば「受動PP＋能動MAP」で、絶対SBP/DBPに初めて手が届く

## 補足（質疑用メモ・スライド外）

- スライド2/3は **3手法 × {Direct, B+T} = 6系列**を提示。B+Tは絶対値ではむしろ悪化するが、その悪化はレベル(between)＝MAP誤差に集中しており、「MAP≈0／PP正」の構図は6系列すべてで一貫
- 過去研究は否定ではなく踏み台：RTBP=従来基準、SinBP_M=PP源の発見、SinBP_D=正直に弱かったと報告
- SinBP_Dの過去主張は centered誤差（トラッキング誤差の小ささ）の話で「絶対値」の主張ではない＝撤回不要
- 数値はすべて 12被験者・2,467拍・LOSO・20秒窓の解析出力に基づく（mean±SD, SDは被験者間）
- n=12で被験者間・絶対値は統計的に非力。押し込み・RGBは新データ収集が前提
